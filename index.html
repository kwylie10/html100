Sementic Elements

The semantic element clearly defines its content or meaning. Developers who use semantic elements will create a distinguishable way to allow other developers to identify location, content, and styling.

There are some styling elements that can be used in HTML vs CSS style sections.
Reasons to Use Semantic Elements
  1. Easily change or update code another developers code.
  2. Easily change your own code.
  3. Provides key words to browers

Non-Sematic Elements

A developer who uses non-semantic elements makes it difficult for other developers to read and interpret their code when making necessary changes or updates to websites or web pages.
